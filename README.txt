Création d’un Projet en utilisant Git et GitHub

Dans la création de ce projet je vais utiliser plusieurs outils :

- D’abord j’ai crée un dossier pour l’exercice en utilisant la commande « mkdir »
- ensuite je me suis mis sur ce dossier en utilisant « cd »
- j’ai utilisé la commende touch pour créer le Fichier « README.txt »
- j’ai ouvert ce fichier en utilisant mon textedit.
- j’ai ensuite tranquillement écrit ces quelques lignes.
- ensuite j’ai fait un premier commit.
- J’ai crée en suivant le même procédé un fichier text appelé « autre »
- je fais un deuxième commit.
- après avoir rajouté des informations j’ai encore fait un commit.
- pour finir j’ai crée un fichier html appelé « resume »
- que j’ai rempli avec les informations des deux fichier textes.
- j’ai fait un énième commit, pour enregistrer le fichier html.

et voilà.

et non c’est pas fini après j’ai poussé ce travail sur GitHub en suivant les étapes apprises dans le cours.
